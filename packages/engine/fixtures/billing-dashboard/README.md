# Billing Dashboard Proof Pack

This fixture is the deterministic demo path for the AI UI acceptance pilot.

- `screens/broken.ui.json` must fail `userface guard --offline --fail-on warning`.
- `screens/fixed.ui.json` must pass the same guard.
- `expected/*` is generated by `pnpm proof-pack:generate` from the real engine CLI output.

The fixture is intentionally small: it proves design-system component contracts,
registry boundary enforcement, zero-upload offline guard behavior, and a buyer-readable
readiness report without requiring a model call or desktop app.
